# Elder Care Assistant

## Overview

An AI-powered assistant that helps elderly people manage medications,
doctor appointments, emergency situations, and daily wellness.

---

## Problem Statement

Many elderly people forget medicines, appointments, or cannot quickly
contact family members during emergencies.

---

## Objectives

- Medication reminders
- Doctor appointment reminders
- Emergency SOS
- Daily wellness check-ins
- Voice interaction

---

## Functional Requirements

### FR1 User Login

Users shall register and login.

### FR2 Medication Reminder

Users receive reminders based on schedules.

### FR3 Appointment Reminder

Users receive appointment notifications.

### FR4 Emergency SOS

One click sends alerts to caregivers.

### FR5 Daily Check-in

Users answer simple wellness questions.

---

## User Stories

As an elderly user,

I want medication reminders

So that I never miss medicines.

As a caregiver,

I want emergency notifications

So that I can respond immediately.

---

## Acceptance Criteria

✓ User can login

✓ Reminder notifications work

✓ SOS reaches caregiver

✓ Appointments are saved

✓ Daily check-in is stored
